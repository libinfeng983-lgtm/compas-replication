# COMPAS Replication Project

## Overview
This project replicates part of the ProPublica analysis on bias in the COMPAS recidivism prediction system.

## Data
The dataset is obtained from the ProPublica GitHub repository.

## Method
We compute false positive rates and false negative rates across racial groups.

## How to Run

pip install -r requirements.txt  
python main.py  

## Output
The script prints error rates for African-American and Caucasian groups.
